# pytorch-slot-filling
基于pytorch的slot filling简单实现，不是很完善，只是一个用于练习的例子，F1在92左右。
数据集只用的ATIS，评测使用的conlleval.pl。

## Requirements
- Python 2.7
- Pytorch 0.10
- numpy
- pickle

## Usage
```python
python main.py
```
